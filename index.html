📖 Guide — Editing the Team Hub

This hub is open to the whole team. Anyone can make changes — no manager login required.

How changes work

  • Edits are local. When you edit the schedule, roster, or time off, changes save to your browser (localStorage) so they persist when you refresh.
  • To share changes with the team, click Export JSON (top right) and share the file (Slack, email, or commit it to the repo as data.json).
  • To load someone else's changes, click Import JSON and pick their file.
  • To start fresh, click Reset to default.

Editing the schedule

  • Click any day cell on the Schedule tab to cycle through: Off → Working → WFH → PTO → Half day.
  • Click + Add Agent to add a new team member. Click the ✎ pencil icon to edit or 🗑 trash to remove.

Time off

  • Click + Add Request on the Time Off tab to log vacation / Jiffy day / etc.
  • Filter by month, supervisor, or type using the dropdowns.
  • Time off automatically updates the "Out Today" list on the Today tab and PTO pills on the Schedule grid.
💡 Tip: Export JSON periodically as a backup. If someone clears their browser data, the local state is lost — but a saved JSON file will restore everything.
